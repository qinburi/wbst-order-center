# 维柏思特订单中心原型

维柏思特订单中心、客户信用管理与客户多端下单联动的可交互评审原型。

## 当前版本

- 版本：`V1.0.0`
- 发布日期：`2026-07-24`
- 状态：基线版本
- Git tag：`v1.0.0`

## 查看方式

直接打开根目录的 `index.html` 即可离线评审。也可以在本目录启动任意静态文件服务后访问该文件。

历史快照位于：

```text
versions/v1.0.0/index.html
```

原型内嵌了 Logo、Iconfont 和 ECharts，不依赖外部 CDN。内部 PC、客户 PC 顶部版本条可打开版本管理页；移动端不展示版本入口。

## 内容范围

- 客户管理及客户主体、账号、授权管理、授权审批列表。
- 订单中心工作台、高密度业务列表和四个流程视图。
- 客户信用工作台、信用管理七个业务页面和三个泳道流程视图。
- 内部 PC、客户 PC、APP/微信小程序移动端联动原型。
- 真实业务弹窗、列表详情、行展开、审批交互和本地演示数据保存。
- 常驻数值图表、红黄绿预警、工作台待办与业务页跳转。

## 版本规则

版本号采用 `V主版本.次版本.修订版本`：

- 主版本：业务架构或整体流程发生重大调整。
- 次版本：新增菜单、页面或完整业务能力。
- 修订版本：页面优化、问题修复和小范围交互调整。

`V1.0.0` 已于 2026-07-24 按当前页面重新建立为基础版本。自本次基线完成后，已发布版本的历史快照和 Git tag 禁止覆盖。发布新版本时应新增快照目录、更新 `CHANGELOG.md`，并创建同号 Git tag。

## 说明

本仓库只包含脱敏后的演示原型与版本资料。页面中的客户、联系人、订单、金额、地址和状态数量均为演示数据，不代表生产数据；标记为 `[待确认]` 的规则仍需业务评审。
