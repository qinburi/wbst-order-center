# 维柏思特订单中心原型

维柏思特订单中心、客户信用管理与客户多端下单联动的可交互评审原型。

## 当前版本

- 版本：`V1.0.0`
- 发布日期：`2026-07-22`
- 状态：基线版本
- Git tag：`v1.0.0`

## 查看方式

直接打开根目录的 `index.html` 即可离线评审。也可以在本目录启动任意静态文件服务后访问该文件。

历史快照位于：

```text
versions/v1.0.0/index.html
```

原型内嵌了 Logo、Iconfont 和 ECharts，不依赖外部 CDN。内部 PC、客户 PC 顶部版本条可打开版本管理页；移动端不展示版本入口。

## 内容范围

- 订单中心菜单、订单工作台和高密度业务列表。
- 客户信用工作台、信用档案、额度、审批、逾期和放行页面。
- 客户授权、订单主流程、订单变更、拒收与退货四张业务流程图。
- 内部 PC、客户 PC、APP/微信小程序移动端联动原型。
- 预订单、正式订单、履约物流、售后、对账回款功能框架。
- 移动首页订单指标、订单进度环形图、快捷下单和订单跟踪。

## 版本规则

版本号采用 `V主版本.次版本.修订版本`：

- 主版本：业务架构或整体流程发生重大调整。
- 次版本：新增菜单、页面或完整业务能力。
- 修订版本：页面优化、问题修复和小范围交互调整。

已发布版本的历史快照和 Git tag 禁止覆盖。发布新版本时应新增快照目录、更新 `CHANGELOG.md`，并创建同号 Git tag。

## 说明

本仓库只包含脱敏后的演示原型与版本资料。页面中的客户、联系人、订单、金额、地址和状态数量均为演示数据，不代表生产数据；标记为 `[待确认]` 的规则仍需业务评审。
